Next.js 15 brings a more stable App Router and improved server components. Here's what matters for building fast sites.

## App Router

The App Router uses a file-system based routing. Each folder represents a route segment. Layouts wrap child segments and persist across navigations.

## Server Components

By default, components in the App Router are **React Server Components**. They run only on the server, so you can fetch data and access backends without exposing secrets to the client.

- No `useState` or `useEffect` in server components.
- Use async/await directly in the component.
- Pass serializable props to client components.

## Client Components

Add `"use client"` at the top when you need interactivity: hooks, event handlers, or browser APIs. Keep the client boundary as low as possible for better performance.

You're now ready to build with Next.js 15.
